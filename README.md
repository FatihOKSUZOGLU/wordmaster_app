# WordMaster - İngilizce Kelime Öğrenme Uygulaması

Modern ve kullanıcı dostu bir Flutter uygulaması.

## Kurulum Adımları

### 1. Projeyi Aç
- VS Code'u aç
- File → Open Folder → `wordmaster_app` klasörünü seç

### 2. Paketleri İndir
Terminal'de şu komutu çalıştır:
```bash
flutter pub get
```

### 3. Çalıştır
- VS Code'un sağ alt köşesinden bir cihaz seç (Chrome önerilir)
- F5 tuşuna bas veya Run → Start Debugging

## Özellikler
✅ Kullanıcı seviye seçimi (A1-C2)
✅ Günlük kelime önerisi
✅ Sözlük ve arama
✅ Detaylı kelime anlamları
✅ Modern ve şık arayüz

## JSON Dosyası
`assets/words.json` dosyasına kendi kelimelerinizi ekleyebilirsiniz.

## Sorun Giderme
- Eğer hata alırsanız: `flutter clean` sonra `flutter pub get`
- Import hataları: Dosya isimlerini kontrol edin
